# SmartMCC Feedback

SmartMCC SW 사용 중 발견한 버그, 개선 요청, 피드백을 등록하는 레포지토리입니다.

## 이슈 등록 방법

1. 상단의 **[Issues](../../issues)** 탭으로 이동합니다.
2. **New issue** 버튼을 클릭합니다.
3. 아래 두 가지 템플릿 중 적합한 것을 선택합니다.

| 템플릿 | 설명 |
|---|---|
| 💬 피드백 / 개선 요청 | 불편한 점, 개선 아이디어, 기능 요청 |
| 🐛 버그 리포트 | 명확한 오류, 크래시, 예상치 못한 동작 |

## 카테고리 안내

| 카테고리 | 설명 |
|---|---|
| 🖥️ UI | 화면 레이아웃, 버튼, 텍스트 등 사용자 인터페이스 |
| ⚙️ 기능 | 기능 오류 또는 동작 이상 |
| 🔗 OV 연동 | OpenVPN 또는 외부 시스템 연동 문제 |
| 📊 데이터 / 설정 | 데이터 처리, 설정값 관련 문제 |
| 🚀 성능 | 속도 저하, 메모리 과다 사용 등 |
| 📋 기타 | 위 카테고리에 해당하지 않는 경우 |

## 자동 동기화

이 레포지토리에 등록된 이슈는 **GitHub Actions**를 통해 [minsko-phd/SmartMCC](https://github.com/minsko-phd/SmartMCC)로 자동 동기화됩니다.

> ⚠️ **관리자 설정 필요**: 동기화를 위해 레포 Settings → Secrets and variables → Actions → `SMARTMCC_SYNC_TOKEN` 등록이 필요합니다.
